Calculation Methodology & Audit Trail

One review reference for pricing basis, component methods, assumptions, and any user overrides.

Material-only basis

Pipe Weight & Rate

Pipe mass formula: W = 0.0246615 x (OD - t) x t.

  • W = pipe mass in kg/m.
  • OD = actual outside diameter in mm from the built-in OD table.
  • t = wall thickness in mm from direct entry or the selected schedule.
  • Total weight = W x length in meter.
  • Pipe Rs/m = finished Rs/kg x pipe kg/m.

Example - 6 IN CS pipe, 7.11 mm wall: W = 0.0246615 x (168.3 - 7.11) x 7.11 = 28.26 kg/m. For 100 m, total weight = 2,826.00 kg.

Material Rate, Factors & P90

  • Raw material Rs/kg is selected from the material library for the chosen year and material standard.
  • Finished pipe Rs/kg = raw material Rs/kg x pipe estimate factor.
  • Normal total is the working estimate; P90 is the higher budget envelope.
  • Overrides replace the relevant default only and are recorded below when used.

Example - 6 IN CS pipe, 2026: raw material Rs 56.50/kg x normal pipe factor 1.80 = finished rate Rs 101.70/kg. x 28.26 kg/m = Rs 2,874.41/m.

Fittings, Gaskets & Strainers

  • A matching-pipe basis is calculated from size and the applicable schedule or thickness.
  • Unit Rs = equivalent pipe rate x approved component factor x applicable pressure and material multipliers.
  • Normal total = unit Rs x BOM quantity; P90 uses the active P90-to-normal ratio.
  • Generic fallback factors are visibly marked and should be validated by quotation.

Example - 6 IN CS 90 degree elbow, 150#: matching pipe rate Rs 2,874.41/m x elbow factor 0.55 x pressure multiplier 1.00 = Rs 1,580.93 per No.

Flange Weight Model

  • Recognised WN, SO and Blind flanges use the flange-weight-3-input-model-v2 JSON model.
  • Flange weight is matched by flange type, pressure rating, and nominal size.
  • Unit Rs = flange weight x material Rs/kg x P50 base multiplier x material multiplier.
  • Where a type-specific lookup is unavailable, the app records the WN fallback used or requests review.

Example - 6 IN WN flange, 150#, CS: JSON weight 10.60 kg x raw material Rs 56.50/kg x WN multiplier 3.70 = Rs 2,215.93 per No.

Valve Pricing Method

  • Gate valves use the built-in size-band weight equations and conversion factors.
  • Other recognised valve types use the approved ratio relative to the gate-valve base for the same size and rating.
  • Pressure class and material multiplier are applied where available.
  • Rows without a usable size, rating, material, or factor remain marked Review.

Example - 6 IN gate valve, 150#, CS: weight basis = 2.00 x 6 squared = 72.00 kg. 72.00 x Rs 56.50/kg x conversion factor 3.70 = Rs 15,058.80 per No.

Stud Bolt & Nut Set Method

  • Readable metric designations such as M30 x 200 use one fully threaded stud plus two heavy-hex nuts.
  • Mass uses standard coarse pitch, effective thread diameter, heavy-hex nut dimensions, and 7,850 kg/m3 density.
  • Unit Rs = complete set mass x material Rs/kg x commercial factor 2.50.
  • Unreadable bolt rows retain the existing review or fallback route; washers, coating and packing are excluded.

Example - M30 x 200 stud set: stud plus two heavy-hex nuts = 1.66 kg/set. 1.66 kg x Rs 56.50/kg x commercial factor 2.50 = Rs 234.88 per set.

Material Classification & BOM Review

  • Materials are classified using astm_piping_material_specification_webapp.json and its Basic Material of Construction logic.
  • BOM items are grouped as Pipe, Fitting, Flange, Valve, Bolt, Gasket, Trap/Strainer, or Other.
  • Only recognised rows with sufficient data receive a calculated rate.
  • Rows marked Review need a confirmed rate, missing data, or a management decision before reliance on the group total.

Example - material detection: a 6 IN item with ASTM A106 Gr.B is classified as Carbon Steel; ASTM A335 Gr.P5 is classified as Low & Intermediate Alloy Steel.

Coating & Scope

  • Coating applies to pipe values only: Yes means internal epoxy lining, external PE coating, or both.
  • Coating Yes factor = 2.30 normal / 3.80 P90; Coating No factor = 1.80 normal / 2.70 P90.
  • Material category, raw rate, factor, P90, and component pricing are indicative cost-estimation inputs.
  • Validate SS, alloy, duplex, non-ferrous, and commercial rates with supplier quotations.

Example - 6 IN pipe coating: the same pipe uses factor 1.80 when non-coated and 2.30 when coated. Coating changes pipe values only; it does not change fitting or valve factors.
